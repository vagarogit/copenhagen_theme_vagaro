Mega Menu with Radix-Style Transitions

This example demonstrates how to use the new Radix-style transitions for your mega menus. The transitions include smooth scale-in/out animations, directional slide animations, and proper state management with data attributes.

How It Works

CSS Classes Used:

  • .mega-menu-full-width - Full-width container with slide animations
  • .mega-menu-content - Content wrapper with scale animations
  • .mega-menu-item - Individual menu items with hover effects
  • .mega-menu-category-heading - Category titles
  • .mega-menu-arrow - Dropdown arrow indicator
  • .mega-menu-trigger - Trigger buttons with state management

Data Attributes:

  • data-state="open|closed" - Controls animation states
  • data-motion="from-start|from-end|to-start|to-end" - Controls animation direction

Key Features:

  • Smooth scale-in/out animations with 3D perspective
  • Directional slide animations (left/right)
  • Proper state management with data attributes
  • Hover effects with smooth transitions
  • Responsive design with proper breakpoints
  • Accessibility-friendly focus states

Usage Example

// JavaScript usage
const megaMenu = document.getElementById('mega-menu-wrapper');

// Show with animation
megaMenu.style.display = 'block';
megaMenu.setAttribute('data-state', 'open');
const content = megaMenu.querySelector('.mega-menu-content');
content.setAttribute('data-state', 'open');
content.setAttribute('data-motion', 'from-start'); // or 'from-end'

// Hide with animation
megaMenu.setAttribute('data-state', 'closed');
content.setAttribute('data-state', 'closed');
content.setAttribute('data-motion', 'to-start'); // or 'to-end'
setTimeout(() => {
    megaMenu.style.display = 'none';
}, 200);
